Arivu API Documentation

Programmatic access to research ancestry graphs, DNA profiles, intellectual mutations, and more. All endpoints require authentication via API key.

Base URL: https://arivu.app/v1
Endpoints
  • GET /v1/papers/search
  • GET /v1/papers/{id}/graph
  • GET /v1/papers/{id}/dna
  • GET /v1/papers/{id}/score
  • POST /v1/papers/{id}/prune
  • GET /v1/papers/{id}/gaps
  • GET /v1/papers/{id}/mutations
  • GET /v1/fields/{name}/fingerprint
  • POST /v1/subscriptions

Authentication

All API requests must include your API key in the X-API-Key header. You can generate API keys from your Account Settings page. 

curl -H "X-API-Key: your_api_key_here" \
     https://arivu.app/v1/papers/search?q=attention+transformer

Requests without a valid API key will receive a 401 Unauthorized response. Keys are tied to your account tier and inherit its rate limits.

Endpoints

GET /v1/papers/search Search for papers ▼

Search for academic papers by title, DOI, or keywords. Returns matching papers from Semantic Scholar and OpenAlex.

Query Parameters
NameTypeRequiredDescription
q string Yes Search query (min 3 characters). Can be a paper title, DOI, or keywords.
limit integer No Maximum number of results (default: 10, max: 50).
Example Response
{
  "results": [
    {
      "paper_id": "204e3073870fae3d05bcbc2f6a8e263d9b72e776",
      "title": "Attention Is All You Need",
      "authors": ["Ashish Vaswani", "Noam Shazeer"],
      "year": 2017,
      "citation_count": 98234,
      "fields_of_study": ["Computer Science"]
    }
  ],
  "total": 1
}
GET /v1/papers/{id}/graph Get ancestry graph ▼

Retrieve the full intellectual ancestry graph for a paper. Returns nodes, edges with mutation types, and metadata. If no cached graph exists, one will be built (which may take 30-120 seconds).

Path Parameters
NameTypeRequiredDescription
id string Yes Semantic Scholar 40-character corpus ID of the seed paper.
Query Parameters
NameTypeRequiredDescription
max_depth integer No BFS crawl depth (1-3, default: adaptive based on paper age).
max_refs integer No Maximum references per paper (10-50, default: 50).
Example Response
{
  "nodes": [
    {
      "paper_id": "204e3073...",
      "title": "Attention Is All You Need",
      "year": 2017,
      "citation_count": 98234,
      "fields_of_study": ["Computer Science"]
    }
  ],
  "edges": [
    {
      "source": "204e3073...",
      "target": "abc12345...",
      "mutation_type": "generalization",
      "confidence_tier": "HIGH",
      "citation_intent": "methodological_adoption"
    }
  ],
  "metadata": {
    "graph_id": "a1b2c3d4...",
    "seed_paper_id": "204e3073...",
    "seed_paper_title": "Attention Is All You Need",
    "total_nodes": 152,
    "total_edges": 203,
    "model_version": "1.0.0"
  }
}

GET /v1/papers/{id}/score Get intellectual diversity score ▼

Retrieve the intellectual diversity score for a paper's ancestry. Returns a composite score (0-100) based on four components: temporal span, field diversity, mutation diversity, and depth ratio.

Path Parameters
NameTypeRequiredDescription
id string Yes Semantic Scholar 40-character corpus ID.
Example Response
{
  "overall_score": 72.5,
  "components": {
    "temporal_span": 85.0,
    "field_diversity": 63.2,
    "mutation_diversity": 71.8,
    "depth_ratio": 70.0
  },
  "context_note": "Above average diversity for Computer Science papers"
}
POST /v1/papers/{id}/prune Simulate paper removal ▼

Simulate removing one or more papers from history and compute the cascading impact on the ancestry graph. Returns which papers would collapse and which would survive.

Path Parameters
NameTypeRequiredDescription
id string Yes Semantic Scholar corpus ID of the seed (root) paper.
Request Body (JSON)
NameTypeRequiredDescription
paper_ids string[] Yes Array of 40-character corpus IDs to remove from history.
Example Request
POST /v1/papers/204e3073.../prune
Content-Type: application/json

{
  "paper_ids": [
    "abc12345678901234567890123456789012345678"
  ]
}
Example Response
{
  "collapsed_nodes": [
    {
      "paper_id": "def98765...",
      "bfs_level": 1
    }
  ],
  "surviving_nodes": [
    {
      "paper_id": "ghi11111...",
      "survival_path": ["ghi11111...", "jkl22222..."]
    }
  ],
  "impact_percentage": 31.0,
  "collapsed_count": 47
}
GET /v1/papers/{id}/gaps Find research gaps ▼

Identify potential research gaps in the paper's ancestry graph. Uses pgvector-based semantic analysis to find underexplored areas and missing connections between research clusters.

Path Parameters
NameTypeRequiredDescription
id string Yes Semantic Scholar 40-character corpus ID.
Example Response
{
  "gaps": [
    {
      "description": "No papers bridge attention mechanisms with reinforcement learning in this lineage",
      "cluster_a": "Attention Mechanisms",
      "cluster_b": "Reinforcement Learning",
      "semantic_distance": 0.72,
      "confidence": "MEDIUM"
    }
  ],
  "total_gaps": 3
}
GET /v1/papers/{id}/mutations List intellectual mutations ▼

Retrieve all classified intellectual mutations (edges) in the paper's ancestry graph. Each mutation describes how one paper inherited and transformed ideas from another.

Path Parameters
NameTypeRequiredDescription
id string Yes Semantic Scholar 40-character corpus ID.
Query Parameters
NameTypeRequiredDescription
type string No Filter by mutation type: adoption, generalization, specialization, hybridization, contradiction, revival, incidental.
min_confidence string No Minimum confidence tier: HIGH, MEDIUM, LOW, SPECULATIVE.
Example Response
{
  "mutations": [
    {
      "edge_id": "204e3073...:abc12345...",
      "citing_paper": "Attention Is All You Need",
      "cited_paper": "Neural Machine Translation by Jointly Learning to Align",
      "mutation_type": "generalization",
      "confidence_tier": "HIGH",
      "citation_intent": "methodological_adoption",
      "mutation_evidence": "Extended single-head attention to multi-head parallel attention across all layers"
    }
  ],
  "total": 203
}
GET /v1/fields/{name}/fingerprint Get field fingerprint ▼

Retrieve the intellectual fingerprint of an academic field, computed from ancestry graphs built across the field. Shows the dominant mutation patterns, key foundational papers, and structural characteristics of the field's research lineage.

POST /v1/subscriptions Subscribe to webhooks ▼

Create a webhook subscription to receive notifications when specific events occur. Arivu will send HTTP POST requests to your callback URL with event data, signed using HMAC-SHA256.

Request Body (JSON)
NameTypeRequiredDescription
event string Yes Event type to subscribe to: graph.built, graph.updated, paper.retracted.
url string Yes HTTPS callback URL to receive webhook events.
secret string No Optional shared secret for HMAC-SHA256 payload signing. If omitted, the global webhook signing secret is used.
Example Request
POST /v1/subscriptions
Content-Type: application/json
X-API-Key: your_api_key_here

{
  "event": "graph.built",
  "url": "https://your-server.com/webhooks/arivu",
  "secret": "your_signing_secret"
}
Example Response
{
  "subscription_id": "sub_a1b2c3d4",
  "event": "graph.built",
  "url": "https://your-server.com/webhooks/arivu",
  "created_at": "2026-03-16T12:00:00Z",
  "status": "active"
}
Webhook Payload (sent to your URL)
{
  "event": "graph.built",
  "timestamp": "2026-03-16T12:05:00Z",
  "data": {
    "graph_id": "a1b2c3d4...",
    "seed_paper_id": "204e3073...",
    "total_nodes": 152,
    "total_edges": 203
  }
}

// Signature header: X-Arivu-Signature: sha256=...

Rate Limits

Rate limits are applied per API key and depend on your account tier. When rate limited, the API returns 429 Too Many Requests with a Retry-After header indicating seconds until the limit resets.
TierRequests / minuteGraph builds / day
Free 30 5
Researcher 60 25
Developer 120 100
Lab 200 250

Error Codes

StatusCodeDescription
400VALIDATION_ERRORInvalid request parameters.
401AUTHENTICATION_ERRORMissing or invalid API key.
403AUTHORIZATION_ERRORYour tier does not have access to this endpoint.
404PAPER_NOT_FOUNDPaper not found in any data source.
422NO_ABSTRACTPaper has no abstract; cannot build meaningful graph.
429RATE_LIMITRate limit exceeded. Check Retry-After header.
500INTERNAL_ERRORUnexpected server error.
502EXTERNAL_API_ERRORUpstream data source (Semantic Scholar, OpenAlex) unavailable.
503NLP_WORKER_ERRORNLP processing service is temporarily unavailable.

  "error": "PAPER_NOT_FOUND",
  "message": "No paper found with the given ID",
